# flask-pytorch-inference

This is a sample flask application that uses Bootstrap for the Frontend UI and PyTorch for model inference.

## Dependencies

Flask
opencv-contrib-python
bootstrap
torch
torchvision
torchsummary

## How to Use

```bash
python app_basic.py
```

Go to 127.0.0.1:4555 on your browser.
